How it works

End-to-End Detection Pipeline

From raw blood smear image to clinical report — seven precision steps powered by EfficientNet B4 & B7 ensemble deep learning.

🔬
01
Image Acquisition

Microscopic blood smear images captured from ALL-IDB & Kaggle datasets

🖼
02
Preprocessing

Noise removal, colour normalisation and contrast enhancement applied

✂️
03
Segmentation

White blood cells isolated and separated from background tissue

📊
04
Feature Extraction

Morphological, texture and colour features extracted per cell

🧠
05
EfficientNet B4 + B7

Dual EfficientNet models independently classify each cell with confidence scores

⚖️
06
Ensemble Voting

B4 & B7 predictions combined via weighted ensemble for final ALL/AML/CML/CLL classification

📋
07
Clinical Report

Structured diagnostic report generated with specialist recommendation and evaluation metrics

Dataset

Trained on ALL-IDB & Kaggle Datasets

The system leverages two trusted leukemia image databases — the ALL-IDB benchmark and curated Kaggle datasets — providing diverse and well-annotated training data.

  • ✓ ALL-IDB: Benchmark microscopic blood smear images with annotated blast and normal cells
  • ✓ Kaggle: Multi-class leukemia dataset covering ALL, AML, CML and CLL subtypes
  • ✓ Combined dataset used to train, validate and test the classification model
  • ✓ Supports reproducible, benchmarkable model evaluation
  • ✓ Over 10,000+ annotated cell images across all leukemia classes

About the Project

Bridging AI and Healthcare for Better Outcomes

Leukemia is one of the most common blood cancers, and early diagnosis significantly improves survival rates. Traditional manual analysis of blood smears is time-consuming and prone to human error.

This project applies machine learning and computer vision to automate the detection process — reducing diagnosis time, improving accuracy, and making expert-level analysis more accessible to healthcare providers.

The system is not intended to replace medical professionals, but to serve as a powerful assistive tool that enhances their diagnostic capability and confidence.
